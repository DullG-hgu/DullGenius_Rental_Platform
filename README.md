# 🎲 덜지니어스 보드게임 대여소 (DullGenius BoardGameRent)

**덜지니어스 보드게임 대여소**는 동아리 부원들이 보드게임을 쉽고 편리하게 대여/반납하고, 관리자가 이를 효율적으로 운영할 수 있도록 돕는 웹 서비스입니다.
기존 Google Apps Script 기반에서 **React (Vite)** 와 **Supabase** 로 마이그레이션하여 더욱 빠르고 안정적인 서비스를 제공합니다.

---

## ✨ 주요 기능 (Features)

### 🧑‍💻 일반 사용자 (User)

**1. 보드게임 탐색 & 조회**
- **검색 & 필터**: 게임 이름, #해시태그, 인원수, 난이도, 카테고리(TRPG, 전략 등)로 원하는 게임을 빠르게 찾을 수 있습니다.
- **상세 정보**: 게임의 난이도, 장르, 추천 인원, 현재 대여 가능 상태(대여중/대여가능)를 실시간으로 확인할 수 있습니다.
- **트렌드 & 추천**: '요즘 뜨는 게임' TOP 5와 관리자가 추천하는 테마별 게임 목록을 제공합니다.

**2. 대여 시스템**
- **방문 수령 예약 (Dibs)**: 원하는 게임을 30분간 '찜'하여 다른 사람이 빌려가지 못하게 확보할 수 있습니다.
- **간편 대여**: 동아리방 키오스크 또는 모바일 웹에서 간편하게 대여 절차를 진행할 수 있습니다.

**3. 커뮤니티 & 마이페이지**
- **리뷰 작성**: 대여한 게임에 대해 별점과 한줄평을 남겨 다른 부원들과 정보를 공유합니다.
- **입고 요청**: 없는 게임이나 추가 입고가 필요한 게임에 대해 '아쉬워요'를 눌러 관리자에게 요청할 수 있습니다.
- **대여 내역 관리**: 현재 대여 중인 게임과 반납 기한, 과거 대여 이력을 한눈에 확인합니다.

---

### 🛡️ 관리자 (Admin)

**1. 대시보드 (Dashboard)**
- **실시간 현황**: 현재 대여 중인 게임, 연체 현황, 최근 활동 로그를 실시간으로 모니터링합니다.
- **데이터 시각화**: 일별/월별 대여 추이를 그래프로 확인하여 운영 전략을 수립할 수 있습니다.

**2. 게임 및 재고 관리**
- **게임 통합 관리**: 게임 추가/수정/삭제 및 상태(분실/파손/수리중) 관리가 가능합니다.
- **외부 API 연동**: 네이버 쇼핑 검색 또는 BGG 정보를 통해 게임 정보를 손쉽게 불러와 등록할 수 있습니다.

**3. 회원 및 포인트 관리**
- **부원 관리**: 전체 부원 목록 조회 및 정보 수정, 회비/벌금 납부 현황을 관리합니다.
- **포인트 시스템**: 보드게임 대회(매치) 결과에 따라 포인트를 지급하거나 차감할 수 있습니다.

**4. 시스템 설정**
- **메인 추천 관리**: 메인 화면에 노출될 추천 테마와 문구를 자유롭게 커스터마이징합니다.

---

### 🤖 키오스크 모드 (Kiosk)

동아리방에 비치된 태블릿(갤럭시 탭 등)을 위한 전용 모드입니다.
- **무인 대여/반납**: 관리자가 없어도 부원들이 스스로 대여 및 반납 처리를 할 수 있습니다.
- **매치 & 룰렛**: 보드게임 매치 결과를 기록하거나, 벌칙 수행 등을 위한 룰렛 기능을 제공합니다.
- **번인 방지**: 장시간 켜져 있는 화면을 보호하기 위한 화면 보호기 및 번인 방지 로직이 적용되어 있습니다.

## 🚀 시작하기 (Getting Started)

### 1. 설치 (Installation)
프로젝트를 클론하고 의존성 패키지를 설치합니다.
```bash
git clone https://github.com/DullG-hgu/DullGenius_Rental_Platform.git
cd dullgboardgamerent
npm install
```

### 2. 환경 설정 (.env)
프로젝트 루트에 `.env` 파일을 생성하고 Supabase 접속 정보를 입력합니다.
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. 실행 (Run)
개발 모드로 실행합니다.
```bash
npm run dev
```
브라우저에서 `http://localhost:5173` 으로 접속하여 확인합니다.

### 4. 빌드 (Build)
배포를 위해 프로덕션 빌드를 생성합니다.
```bash
npm run build
```
